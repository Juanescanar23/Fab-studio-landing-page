# FAB STUDIO Landing Page

Landing page estática para FAB STUDIO, centrada en arquitectura, identidad visual de marca y presentación de proyectos.

## Archivo principal

- `fabstudio-landing.html`
- `vercel.json`: enruta `/` en Vercel hacia la landing principal

## Contenido del repositorio

- `assets/`: logos, iconos e imágenes organizadas para la landing.
- `CENTRO PEDIATRICO DEL CAUCA/`: material visual del proyecto.
- `Casa Nativa /`: material visual del proyecto.
- `CAMBIOS.md`: registro de cambios implementados.
